# By_Verse - E-commerce Project

Ye ek full-stack e-commerce application hai jise MERN stack ka use karke banaya gaya hai.

## Tech Stack
- **Frontend:** React.js, Vite, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Authentication:** JWT (JSON Web Tokens)

## Project Structure
- `Backend/`: Node.js API logic aur database models.
- `Ecom/`: React.js frontend (Pages, Components, Context).

## Setup Instructions
1. Dono folders (`Backend` aur `Ecom`) mein `npm install` karein.
2. Backend folder mein `.env` file banayein aur `MONGO_URI`, `JWT_SECRET` add karein.
3. `npm run dev` se local server start karein.
